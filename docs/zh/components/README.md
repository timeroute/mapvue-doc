# 组件介绍

## 组件列表

### 核心组件

- VMap 地图组件

### 通用组件

- VMarker 标记点
- VPopup 弹窗
- VSprit 图标图层
- VFog 雾

### 控件组件

- VAttributionControl 归属信息控件
- VNavigationControl 导航控件
- VScaleControl 缩放控件

### 图层组件

- VBackgroundLayer 背景色图层
- VCircleLayer 圆点图层
- VFillExtrusionLayer 面挤压图层
- VFillLayer 面图层
- VHeatmapLayer 热力图层
- VHillshadeLayer 山地图层
- VLineLayer 线图层
- VRasterLayer 瓦片图层
- VSymbolLayer 图标图层

### 数据源组件

- VCanvasLayer Canvas 数据源
- VGeoSource GeoJSON 数据源
- VVectorSource 矢量数据源
- VImageSource 图片数据源
- VVideoSource 视频数据源
- VRasterSource 瓦片数据源
- VRasterDemSource 地形瓦片数据源
